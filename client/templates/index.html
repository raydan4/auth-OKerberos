Authentication Lab 5: OAuth

Cross Your Fingers

1. Send the Username and Password to the server

Use: ian:password should look something like: {"user":"ian","password":"password"}

2. Receive the Encoded Response from the server

The JSON response should be encrypted.

3. Decode & decrypt the Response using password hash

encrypted_response = base64.b64decode(encoded_response)

hashed_password = SHA256(password)

response = aes256_decrypt(encrypted_response, hashed_password)

4. Once Decrypted, get the resulting JSON object

json_blob = json.loads(response)

5. POST the contents of the decrypted JSON object (the token) to the application server

Token should look something like:

{ "token": "" }

6. Handle the response from the application server

The response will be JSON. It will either say:

{"message":"AUTHORIZED","status":200}

or

{"message":"UNAUTHORIZED","status":403}
